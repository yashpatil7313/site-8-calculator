# 🔢 Calculator App - Day 1 Project 8

## 📌 Project Overview

This project is a simple and interactive **Calculator App** created as part of my semester challenge to build 200 websites.

It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🎯 Features

* 🔢 Number Buttons (0–9)
* ➕ Arithmetic Operations (+, -, *, /)
* 🧮 Calculation Function (=)
* ❌ Clear Display Button (C)
* 📱 Responsive Layout
* 🎨 Clean UI Design
